# diurno.app

Site público do Diurno — página inicial e política de privacidade.

Servido pelo GitHub Pages no apex `diurno.app` (ver `CNAME`).
A documentação vive em repositório separado, em https://docs.diurno.app

- `index.html` — home (a landing e o download do desktop crescem aqui)
- `privacidade/index.html` — política de privacidade

A URL https://diurno.app/privacidade está registrada na verificação OAuth do
Google e nas fichas da App Store e Play Store. **Não mudar esse caminho** sem
atualizar os três.
